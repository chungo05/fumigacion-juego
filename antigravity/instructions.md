#  Objetivo del proyecto

Crear un juego web en el que el objetivo es aplastar los bichos que aparecene n pantalla.

El proyecto ya esta creado pero no tiene funcionalidad ni estilos. El objetivo es implementar el diseño y las funcionalidades del juego, siguiendo EXACTAMENTE los diseños presentados en las referencias que serán proporcionadas que se encuentran en la carpeta de "Antigravity" dentro del proyecto.
